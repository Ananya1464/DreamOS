# Chrome Extension Icons

The extension needs PNG icons. You can:

1. **Quick Solution**: Use any 16x16, 48x48, and 128x128 PNG images
2. **Generate Icons**: Use https://www.favicon-generator.org/
3. **Use Emoji**: Convert emoji to PNG at https://emoji.gg/

For now, Chrome will work without icons (it will show a default puzzle piece icon).

## Icon Sizes Needed:
- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels
